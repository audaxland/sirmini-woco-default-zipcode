Sirmini WoCo Default Zipcode is an add-on for WooCommerce, this allows to define a default zipcode and default city for shipping adresses and shipping cost calculation

== Description ==

This plugin is an add-on for WooCommerce, it requires WooCommerce and will not work without.

Sirmini WoCo Default Zipcode allows you to define a default postcode and default city that will be used 
as defaults for the shipping adresses and the shipping cost calculator.

This is mainly usefull to set a default shipping option list when your shipping options are restricted to certain areas
like for a local delivery option that you can restrict to local zipcodes and still display it by default in the list of
available delivery options before the customer has entered his/her postcode.

== Installation ==

Install and activate WooCommerce first.
Upload and activate the Sirmini WoCo Default Zipcode.
Go on the "WooCommerce->Settings" page and select the "Shipping" tab, then scroll down and set a default zipcode and city.
Save you settings and you're done.
